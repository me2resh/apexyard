## Checkout → waitlist (build spec)

Mimic L&E / EllaOla **Add to cart → Checkout** so demand feels real; backend is **waitlist**, not ship-from-stock.

**Recommended Shopify setup:**

1. Two products or variants: `blendavit Toddlers (1–3)` · `blendavit Kids (4+)`.
2. **Pre-order / waitlist** app (e.g. Preorder, Waitlister) or manual workflow: checkout collects **email + market + variant + optional refundable deposit**.
3. Post-purchase page + email: “You’re on the list — we’ll confirm when manufacturing batch is ready.”
4. **No false “in stock” shipping promises** — cart copy: “Reserve your box” / “Join the first production run.”
5. Multi-currency: **SAR (KSA)** + **USD (USA)** — Shopify Markets.
6. **Arabic:** Prioritize for KSA (RTL can be Phase 2 if MVP English-first with SAR checkout).
7. **Deposit toggle:** Launch with configurable deposit via pre-order app; operator can switch primary CTA to **$0 reserve** anytime to maximize funnel activity, then re-enable deposit when ready to **secure funds** post–proof-of-concept.
